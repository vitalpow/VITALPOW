¿Qué es VitalPow?

Resumen del proyecto

VitalPow es un arnés inteligente para perros que monitorea signos vitales (temperatura corporal, frecuencia cardíaca y señales de estrés) en tiempo real mediante sensores y un microcontrolador, con alertas luminosas y sonoras para proteger la salud y el bienestar de la mascota.

Integrado desde la documentación técnica y el proyecto universitario, VitalPow busca promover un acompañamiento responsable y anticiparse a riesgos en la salud canina mediante automatización aplicada.

Fuente: documentación del proyecto (PDF integrado en esta página).

Objetivos principales

  • Diseñar un arnés capaz de monitorear la salud del perro con sensores biométricos.
  • Implementar circuitos y microcontroladores dentro de un arnés funcional.
  • Programar alertas inmediatas (luminosas/sonoras) y transmitir datos a una app.
  • Estimar costos de producción y escalabilidad del dispositivo.

Tecnología y componentes

Sensores

Sensor de temperatura de contacto, sensor de pulso/PPG para frecuencia cardiaca, acelerómetro 3-ejes para actividad y detección de jadeo.

Procesamiento y conectividad

Microcontrolador (por ejemplo ESP32/STM32), conectividad Bluetooth/Wi-Fi para enviar datos a la app y almacenamiento local temporal.

Alertas y UX

Buzzer/LEDs para alertas locales, App móvil para notificaciones, interfaz simple para dueños y módulo de respaldo para emergencias.
